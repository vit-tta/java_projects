Задача: Решить квадратное уравнение ax² + bx + c = 0

Решение:
1. Находим дискриминант: D = b² - 4ac
2. Если D > 0, то два корня:
   x₁ = (-b + √D) / (2a)
   x₂ = (-b - √D) / (2a)
3. Если D = 0, то один корень:
   x = -b / (2a)
4. Если D < 0, то действительных корней нет

Пример:
Уравнение: 2x² - 5x + 3 = 0
a = 2, b = -5, c = 3
D = (-5)² - 4*2*3 = 25 - 24 = 1 > 0
x₁ = (5 + √1) / 4 = (5 + 1) / 4 = 6/4 = 1.5
x₂ = (5 - √1) / 4 = (5 - 1) / 4 = 4/4 = 1

Ответ: x₁ = 1.5, x₂ = 1Test content for file
